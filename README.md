# Backend (Node.js + Express + Sequelize)

## Setup
1. Copy `.env.example` to `.env` and fill DB credentials.
2. `npm install`
3. Create MySQL DB: `CREATE DATABASE m2g_ecom;`
4. Run seed: `npm run seed` (this runs node src/seeders/seed.js)
5. Run dev: `npm run dev`

Admin user after seed: admin@example.com / admin123
